# ACM REP 2026

This is the repository for ACM REP 2026 submission "Measuring Ease of Reuse for Security Research Artifacts with Results-Reproduced Badges". We provide our Excel file, containing details for evaluated artifacts. We unfortunately ran out of time, pre-submission, to clean and compile a full set of topologies and scripts we used in this work. We will do this if the paper is accepted, so that our work can be reproduced in full by future researchers.

File results-per-artifact.csv contains our evaluation results per each artifact. File processdetails.py processes those to produce the summary table 1 and other results that we report in the paper.
